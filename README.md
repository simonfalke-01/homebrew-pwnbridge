# Homebrew tap for pwnbridge

Install [pwnbridge](https://github.com/simonfalke-01/pwnbridge) on macOS:

```sh
brew install simonfalke-01/pwnbridge/pwnbridge
```

The formula installs the native macOS client, the Linux x86-64 remote agent,
shell completions, and Mutagen. The remote agent is deployed automatically over
SSH when pwnbridge first connects; nothing needs to be installed manually on the
remote host.
